pySCATMECH : A Python Interface to the SCATMECH Library¶

pySCATMECH is a Python interface to the SCATMECH library of scattering codes. The documentation for the specific models are available at SCATMECH Polarized Light Scattering C++ Class Library.

To install, use pip install pySCATMECH at a command prompt.
